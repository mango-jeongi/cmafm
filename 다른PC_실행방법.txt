====================================================
  CMAFM Dashboard — 다른 PC 실행 방법
====================================================

[ 필요한 것 ]
  - Windows 10/11 (64bit)
  - Python 3.11 이상 (없으면 아래에서 설치)
  - 이 폴더 전체 (모델 파일 포함)

====================================================
  STEP 1 : Python 설치 (처음 1회만)
====================================================

  아래 주소에서 Python 다운로드 후 설치:
  https://www.python.org/downloads/

  ★ 설치 화면에서 반드시 체크!
     [v] Add Python to PATH

====================================================
  STEP 2 : 패키지 설치 (처음 1회만)
====================================================

  install.bat 파일을 더블클릭하여 실행
  (자동으로 필요한 패키지를 모두 설치합니다)

  GPU가 있으면 CUDA 버전, 없으면 CPU 버전 자동 선택

====================================================
  STEP 3 : 대시보드 실행
====================================================

  run_dashboard.bat 파일을 더블클릭

  터미널 창에 아래 메시지가 뜨면 성공:
  "Local URL: http://localhost:8501"

  브라우저에서 http://localhost:8501 접속

====================================================
  STEP 4 : 탐지 실행
====================================================

  1. 왼쪽 사이드바 → "모델 로드" 버튼 클릭
  2. 탭 선택:
     - 이미지 탐지: RGB + Thermal 이미지 업로드
     - 영상 탐지:   RGB + Thermal 영상 업로드
     - 샘플 테스트: 데이터셋 자동 로드 (데이터 있을 때)

====================================================
  주의사항
====================================================

  - 모델 파일(runs/best.pth)이 반드시 있어야 함
  - CPU만 있는 경우 추론이 느릴 수 있음 (이미지 1장 ~5~10초)
  - 영상 업로드 최대 500MB
  - 이미지는 jpg, png, bmp 지원
  - 영상은 mp4, avi, mov 지원
